# Project

This is a Distributed Lag Non-Linear Model project in R. Our goal is to make this model more efficient and benchmark it at different scales.

## Language

This project is written in R. But we will be looking to rebuild functions in Rust in order to improve performance and scalability.

## Tools

We will be using the following tools:

- R for data analysis and modeling
- Rust for performance optimization and scalability
- Python for data preprocessing and visualization
